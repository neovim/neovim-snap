<p align="center"><b>This is the snap for nvim</b>. It works on Ubuntu, Fedora, Debian, and other major Linux
distributions.</p>

## Install

### Stable release (0.10.2)

```sh
sudo snap install nvim --classic
```

### Nightly release 
```sh
sudo snap install nvim --edge --classic
```
